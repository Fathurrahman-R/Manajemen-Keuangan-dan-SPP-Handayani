Each app boots independently from its own directory: `cd backend && php artisan serve` for the API, `cd frontend-v2 && php artisan serve` for the Filament panels, and `cd frontend && npm run dev` for the React SPA. Shared DB credentials must be mirrored in `backend/.env` and `frontend-v2/.env`; `frontend-v2/config/handayani.php` points at the backend base URL used by `ApiService`.
