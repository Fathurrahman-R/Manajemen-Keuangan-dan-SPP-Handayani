The root is a multi-app monorepo with three independent Laravel/Node applications sharing a single database. `backend/` owns the REST API, Eloquent models, migrations, seeders, events/listeners, queued jobs, and the Midtrans client; it exposes its contract via `routes/api.php` and documents endpoints under `docs/*.json`. `frontend-v2/` (Filament v4 + Livewire) is the primary operational UI and calls the backend through `app/Services/ApiService`, authenticating via Filament's Sanctum flow. `frontend/` (Vite + React) is a legacy SPA targeting the same API. `portal-reference/` is an unrelated TanStack Start marketing site. Cross-cutting concerns — RBAC (spatie/laravel-permission), role constants (`app/Constant/*`), permission binding (`PermissionBinding`), notifications, exports, imports, and Midtrans — live in the backend so both frontends consume a single source of truth. Feature specs under `.kiro/specs/<feature>/` drive development across all three apps.
